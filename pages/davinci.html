DaVinci Mobile

DaVinci Mobile is LinkedIn's pattern library for designers and developers.

Davinci Mobile originally was a linked PDF. Although the PDF allowed designers to access the information they needed, the setup didnâ€™t scale well and provided limited resources for developers. The solution was to create an online pattern library which fit in with its preexisting sister branches, but also established its own style. Mobile DaVinci is fully-responsive so designers can see components in context on a mobile device.

Although I was originally slated to solely work on design, I volunteered to expedite the project by developing the hub page. Throughout the project I worked closely with designers, web developers, and a fellow intern who implemented my component page designs.

We explored version 1, but after conducting some user testing, we realized version 2 was easier to parse and navigate on a mobile device. This design decision also made for a cleaner development practice: no need for different image assets for mobile!
